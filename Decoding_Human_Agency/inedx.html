Project summary

We investigate the minimal temporal window in which voluntary actions remain controllable ‚Äî the behavioural and neural "point of no return". Using low-latency EMG onset detection, Random Dot Motion tasks, and closed-loop EEG/EMG control, we quantify when actions become irreversible and how that maps to neural signatures.

Goals

  • Define behavioural timing thresholds for action veto (point of no return).
  • Provide reproducible low-latency detectors (RealTimeOnsetDetector) and configs.
  • Publish experiment code (PsychoPy + LSL), timing validation scripts, and example data.

Methods & components

  • Random Dot Motion (RDM) perceptual decision tasks with closed-loop evidence termination.
  • High-precision EMG onset detection (sub-10 ms latency target) and latency benchmarking.
  • Timing validation using photodiode / LSL logs and parallel-port stamps.
  • Real-time data stream: PsychoPy ‚Üî LSL ‚Üî logging & analysis (Python).
